# 🎬 影片色彩追蹤系統

一個純粹使用 HTML、CSS、JavaScript 建構的網頁應用，無需前端框架。

## ✨ 主要功能

- 📁 **上傳影片**：支援本機影片檔案上傳
- 🎯 **選取畫面點位**：點擊影片畫面選擇要追蹤的像素位置
- 🌈 **顯示顏色**：即時讀取該位置的 HEX 與 RGB 顏色
- ⏱️ **顏色變化記錄**：錄製影片播放期間的顏色變化
- 📄 **下載記錄**：匯出紀錄為 CSV 檔案
- 📱 **響應式介面**：桌機、平板、手機皆適用

## 🧩 技術要點

- HTML：清晰的結構和語義化標籤
- CSS：現代卡片式設計、漸層背景、響應式佈局
- JavaScript：DOM 操作、檔案上傳、Canvas 影格讀取、顏色抽取、時間記錄

## 🚀 使用方式

1. 打開 `index.html`。
2. 上傳影片檔案或拖曳影片到區域。
3. 影片載入後，在影片畫面點選希望追蹤的像素位置。
4. 點擊「開始記錄」，影片播放時系統會記錄該位置顏色變化。
5. 點擊「停止記錄」，可選擇下載記錄為 CSV。

## 📁 檔案結構

```
homework-01/
├── index.html      # 主頁面
├── styles.css      # 介面樣式
├── script.js       # 核心邏輯
└── README.md       # 專案說明
```

## 🎯 介面說明

- 左側：影片上傳、顏色資訊、紀錄控制按鈕
- 右側：影片預覽區與記錄時間軸
- 點選影片：顯示選取座標與實際顏色結果

## 📌 特色

- 直接讀取本機影片影格，不需伺服器支持
- 可以追蹤影片播放中的單一像素顏色變化
- 紀錄輸出簡單易用，方便後續分析
